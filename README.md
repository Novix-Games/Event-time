# יומן מדידות שדה

אפליקציית PWA (Progressive Web App) לתיעוד מדידות שטח: שעון בזמן אמת, רישום אירועים עם זמן התחלה אוטומטי, הקלטת שמע, ניהול מספר מדידות, וייצוא לאקסל. הנתונים נשמרים באחסון המקומי של הדפדפן (IndexedDB) ונשארים גם אחרי סגירת האפליקציה.

## מבנה הקבצים

```
field-log-pwa/
├── index.html          ← האפליקציה עצמה
├── manifest.json        ← הגדרות PWA (שם, אייקונים, צבעים)
├── service-worker.js     ← תמיכה בעבודה אופליין + התקנה למסך הבית
├── icons/
│   ├── icon-192.png
│   ├── icon-512.png
│   ├── icon-192-maskable.png
│   └── icon-512-maskable.png
└── README.md
```

## איך להעלות ל-GitHub Pages

1. **צור repository חדש** בחשבון ה-GitHub שלך (למשל `field-log`), ציבורי.
2. **העלה את כל התוכן** של תיקיית `field-log-pwa` (כולל תיקיית `icons`) לשורש ה-repository — דרך "Add file → Upload files" באתר GitHub, או דרך git:
   ```bash
   git clone https://github.com/<username>/field-log.git
   cd field-log
   # העתק לכאן את כל הקבצים מתיקיית field-log-pwa
   git add .
   git commit -m "Field log PWA"
   git push
   ```
3. **הפעל GitHub Pages**: ב-repository → Settings → Pages → תחת "Build and deployment" בחר Source: **Deploy from a branch**, Branch: **main**, תיקייה: **/ (root)** → Save.
4. תוך דקה־שתיים האפליקציה תהיה זמינה בכתובת:
   ```
   https://<username>.github.io/field-log/
   ```
   (בהתאם לדפוס שכבר יש לך ב-`novix-games.github.io`).

**חשוב:** GitHub Pages מגיש הכול תחת HTTPS אוטומטית — זה קריטי, כי הרשאת מיקרופון (`getUserMedia`) עובדת רק בדפים מאובטחים (HTTPS).

## התקנה למסך הבית (כמו אפליקציה אמיתית)

לאחר שהאתר עולה:
- **אנדרואיד (Chrome):** פתח את הכתובת → תפריט (⋮) → "התקן אפליקציה" / "הוסף למסך הבית".
- **iPhone (Safari):** פתח את הכתובת → כפתור השיתוף (□↑) → "הוסף למסך הבית".

לאחר ההתקנה האפליקציה תיפתח במסך מלא בלי סרגל כתובת של הדפדפן, עם אייקון משלה.

## מה משתנה לעומת הגרסה שרצה בתוך Claude

- **שמירה אוטומטית:** עברה מ-API הפרטי של Claude ל-IndexedDB אמיתי בדפדפן — עצמאי לגמרי, לא תלוי בשום שירות חיצוני.
- **הקלטות שמע:** עכשיו **נשמרות לצמיתות** במכשיר (לא רק בזיכרון הזמני של ההפעלה הנוכחית), כי אין יותר את מגבלת ה-sandbox של Claude.
- **הרשאת מיקרופון וייצוא לאקסל:** אמורים לעבוד באופן תקין ואמין, כי הדף רץ כדף עצמאי בדפדפן ולא בתוך iframe מוגבל.
- **גיבוי ידני** (כפתור בדף הבית) עדיין קיים כרשת ביטחון נוספת, ומוריד קובץ JSON עם כל המדידות (לא כולל קובצי השמע עצמם — אלה מורדים בנפרד מכל רשומה).

## עדכון עתידי

כדי לעדכן את האפליקציה בעתיד: פשוט העלה גרסה חדשה של `index.html` (ואולי `service-worker.js`, אם שונה) ל-repository — GitHub Pages יתעדכן אוטומטית תוך דקה־שתיים. שווה גם לשנות את `CACHE_NAME` בתוך `service-worker.js` (למשל מ-`field-log-v1` ל-`field-log-v2`) כדי לוודא שמשתמשים שכבר התקינו את האפליקציה יקבלו את הגרסה החדשה ולא גרסה ישנה מהמטמון.
